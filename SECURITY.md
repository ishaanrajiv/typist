# Security Policy

## Reporting a vulnerability

If you discover a security issue, do not open a public issue.

Please report it privately to the maintainer through GitHub private channels.
Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

You can expect an initial response within 3 business days.

## Supported versions

Only the latest `main` branch state is actively supported for security fixes.
